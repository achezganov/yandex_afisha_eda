# Анализ лояльности пользователей Яндекс Афиши
## Описание проекта
В данном проекте проводится исследовательский анализ данных сервиса Яндекс Афиша с целью изучения поведения пользователей и выявления факторов, влияющих на их возврат и повторные покупки.

Результаты анализа помогут:
- выявить пользователей с высокой вероятностью повторных покупок;
- разработать персонализированные предложения;
- повысить эффективность маркетинговых кампаний;
- оптимизировать маркетинговый бюджет;
- увеличить общий уровень удержания клиентов.

## Цель проекта
Провести исследовательский анализ данных и определить факторы, влияющие на повторные заказы пользователей.

## Задачи проекта
Проект включает следующие этапы:
### 1. Подключение к базе данных
- Подключение к базе данных SQL
- Первичное знакомство со структурой данных
- Выполнение SQL-запросов для изучения данных

### 2. Выгрузка данных
- Написание SQL-запроса для выгрузки необходимых данных
- Импорт данных в pandas DataFrame

### 3. Предобработка данных
- Проверка типов данных
- Обработка пропущенных значений
- Поиск и устранение дубликатов
- Подготовка данных к анализу

### 4. Исследовательский анализ данных (EDA)
- Формирование профиля пользователя
- Анализ распределений признаков
- Анализ поведения пользователей
- Анализ повторных заказов
- Поиск закономерностей и зависимостей

### 5. Корреляционный анализ
- Вычисление коэффициентов корреляции
- Использование коэффициента phik
- Определение факторов, влияющих на количество заказов

### 6. Формулирование выводов и рекомендаций
- Интерпретация результатов анализа
- Подготовка рекомендаций для маркетинговой команды
